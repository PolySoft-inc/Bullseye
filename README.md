## Setup Instructions

1. Clone the repository:

```bash
> git clone https://github.com/PolySoft-inc/Bullseye.git
> cd Bullseye
```

2. Install dependencies:

```bash
> bun i
```

3. Run the development server:

```bash
> bun run dev
```
